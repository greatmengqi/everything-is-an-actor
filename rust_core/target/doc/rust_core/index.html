Rust-native Actor core for Python with async handler support

Key insight: Rust mailbox is fast (4.5M msg/s) but PyO3 bridge adds overhead. Full performance requires Rust handler, but we can still get 2-3x improvement by keeping the actor loop in Rust and only calling Python for actual work.
